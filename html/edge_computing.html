邊緣計算 ( Edge Computing )

邊緣計算是一種分散式運算的架構，將應用程式、數據資料與服務的運算，由網路中心節點，移往網路邏輯上的邊緣節點來處理。邊緣運算將原本完全由中心節點處理大型服務加以分解，切割成更小與更容易管理的部份，分散到邊緣節點去處理。邊緣節點更接近於使用者終端裝置，可以加快資料的處理與傳送速度，減少延遲。在這種架構下，資料的分析與知識的產生，更接近於數據資料的來源，因此更適合處理巨量資料。網路邊緣的線上設備數量迅速增加，這些設備產生大量數據資料(例如智慧工廠或城市的IoT設備)，也消耗大量資料頻寬(例如使用5G手機查看 4K 影像或VR 遊戲裝置)，或兩者同時使用(如自動駕駛車輛)。

六大科技領域閱讀心得徵文 活動說明

壹、活動宗旨：

數位科技如AI、區塊鍊及雲計算等快速發展，改變了人們的生活模式，導致未來生活必須與數位科技深度融合。
書香文化教育基金會(以下簡稱本會)成立以來，一直以提昇教育水準與閱讀素養為宗旨，為配合數位科技的發展趨勢，特規劃「六大科技領域閱讀心得徵文活動」，希望透過科技閱讀，使高中職生與大專學生能對新興科技有基本的認識與瞭解，開啟學子們對數位科技的興趣，促使年輕一代對未來有更多想像，能善用數位科技，以創造更美好的生活。

二、蒐集目的：辦理有關六大科技領域閱讀心得徵文活動之參考及後續評選活動等相關程序之聯繫使用。
三、個人資料類別：包含姓名、手機及電子信箱等個人資料中之識別類訊息。
四、個人資料利用之時間、地區、對象及方式：

智慧住家

系統不必傳送與接收主要儲存樞紐的資料，智慧住家只需採用邊緣技術就能具備更大彈性。這將減少延遲、提供更高安全性及降低空載傳輸成本。語音命令將有如即時般更快速發揮作用。

病患監控

邊緣運算有許多機會可應用於住院方面。首先，這代表第三方儲存系統的需求更少。透過在本機處理資料，病患將擁有更多隱私。這也代表能更有效地通知病患的生命徵象。另外，這建立了一目了然的病患設定畫面。

智慧醫療

像是遠距診療、遠距手術、導引機器人、虛擬實境看診。

自動駕駛

人類之所以還無法完全仰賴自駕系統，是因為系統必須做出閃電般快速，而且100％準確的決策。如果有孩子突然衝到街上，在系統傳達指令到讓車子剎車之前是不允許有任何延遲。

自動駕駛

由於讀取更加頻繁，顯然更需要智慧型系統。邊緣運算可藉由最佳化城市內的交通、在必要時開放道路以提供協助，並對於管理無人自駕車而言至關重要。
